# How to Void/Cancel Transaction

## <mark style="color:orange;">**CANCEL WHOLE TRANSACTION**</mark>

1. Click on Transactions

<figure><img src="../.gitbook/assets/Click on Transactions (2).png" alt="" width="563"><figcaption></figcaption></figure>

2. Transaction page

<figure><img src="../.gitbook/assets/Transaction page.png" alt="" width="563"><figcaption></figcaption></figure>

3. Select transaction no you want to cancel then, Click on Cancel

<figure><img src="../.gitbook/assets/Select transaction no you want to cancel then, Click on Cancel.png" alt="" width="563"><figcaption></figcaption></figure>

4. \* You can add note/reason for cancelling transaction (optional) \* Check RETURN STOCKS means stocks under that transaction may return; uncheck means stocks in the transaction will remain deducted

{% hint style="info" %}
\* Check MARK AS UNPAID means the transaction will be unpaid or refunded while uncheck means transaction will remain paid
{% endhint %}

<figure><img src="../.gitbook/assets/You can add note.png" alt="" width="563"><figcaption></figcaption></figure>

5. Click on Cancel

<figure><img src="../.gitbook/assets/5.Click on Cancel.png" alt="" width="563"><figcaption></figcaption></figure>

6. Transaction has been cancelled

<figure><img src="../.gitbook/assets/6.Transaction has been cancelled.png" alt="" width="563"><figcaption></figcaption></figure>
